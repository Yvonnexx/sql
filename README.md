Database Application
=======================

## Tools to install 
1. lxml
*   type in the command line
    ``` 
    sudo pip install lxml

    ```
2. flask
*   type in the command line
    ```
    sudo pip install virtualenv
    mkdir myproject
    cd myproject
    virtualenv venv
    . venv/bin/activate
    venv\scripts/activate

    ```
*  now install flask
    ```
    pip install Flask

    ```

3. sqlalchemy

*   type in the commmand line
    ```
    pip install SQLAlchemy

    ```
