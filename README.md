Database Application

## Tools to install 
1. lxml
    ``` 
    sudo pip install lxml

    ```
2. flask
    ```
    sudo pip install virtualenv
    mkdir myproject
    cd myproject
    virtualenv venv
    . venv/bin/activate
    venv\scripts/activate

    ```

  now install flask

    ```
    pip install Flask

    ```

3. sqlalchemy

    ```
    pip install SQLAlchemy

    ```
